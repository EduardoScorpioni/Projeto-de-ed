# ED Trabalho Prático 2 - Grupo 6

Base inicial em HTML e CSS para o sistema web de ensino de Estrutura de Dados.

## Como abrir

Abra o arquivo `index.html` no navegador.

## Pastas

- `assets/css`: estilos do site.
- `assets/img`: diagramas usados nas páginas.
- `pages`: páginas internas de conteúdo.
- `docs`: anotações e resumo do enunciado.

## Conteúdo inicial

- TAD.
- Lista simplesmente encadeada.
- Lista duplamente encadeada.
- Exemplos em C#.
